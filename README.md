# FrontRowTunes

Since FrontRow was removed in Mac OS X 10.7 Lion, this open-source app aims to bring back a FrontRow-like full-screen music experience, but for music playing in the Music app.

## Features

- Show currently playing track
- Play/pause, next/previous track
- Display options: e.g. position bar, position as mm:ss
- Clock (analog or digital)
- Press (i) in-app to view how to invoke each function
